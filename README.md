# online-courses-test

Тестовый каталог с поиском и фильтрами.

**Описание**
- **online-courses-test**: простая статическая демонстрация страницы курса с наборами карточек в папке `assets/cards`.
- От себя добавлено поднятие карточки по наведению курсора.

**Структура проекта**
- **`index.html`**: входная HTML-страница.
- **`script.js`**: JavaScript-логика страницы.
- **`style.css`**: стили проекта.
- **`assets/`**: медиа и статические ресурсы.

**Как запустить (локально)**
- Откройте `index.html` напрямую в браузере (двойной клик или `File → Open`).
- Или через простой статический сервер:

```bash
# из корня проекта
python3 -m http.server 8000
# затем открыть http://localhost:8000
```

Или с `http-server` (Node.js):

```bash
npx http-server -p 8000
# затем открыть http://localhost:8000
```

**Требует доработки**
- Доработать тени при наведении курсора на карточку.
- Добавить background-элементы страницы.
